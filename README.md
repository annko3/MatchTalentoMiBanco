# Match Talento Mi Banco

**Match Talento Mi Banco** es una plataforma que conecta el talento de los postulantes con las oportunidades laborales ideales.

---

## 🌐 Características principales

* 🧁 Página principal con información de Mi Banco
* 📚 Sección dinámica de vacantes de empleo
* 💬 Diseño responsive y moderno
* 🎨 Paleta de colores personalizada mediante un sistema de **temas globales**

---

## 🛠️ Tecnologías utilizadas

* **Boostrap 5** – Estilos y sistema de diseño
* **Tailwind CSS** – Estilos y sistema de diseño
* **FireBase** – Backend (autenticación, Firestore)
* **Git & GitHub** – Control de versiones
